# Employee Salary Audit Automation (PL/SQL)

## Project Overview
This project demonstrates how to update employee salary using a PL/SQL procedure and automatically track salary changes using a trigger.

## Concepts Used
- SQL Tables
- PL/SQL Procedure
- Trigger
- Audit Logging

## Tables Created
- emp_details (stores employee details)
- salary_audit (stores salary update history)

## Output
Whenever salary is updated, the old and new salary values are automatically inserted into the audit table.
